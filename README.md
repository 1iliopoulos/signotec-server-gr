# signoPAD-API/Web – Ελληνική Έκδοση 🇬🇷

Web εφαρμογή για **λήψη ηλεκτρονικής υπογραφής μέσω συσκευών signotec**, βασισμένη στο επίσημο **signoPAD-API/Web**, πλήρως μεταφρασμένη στα ελληνικά και προσαρμοσμένη για χρήση σε τοπικό περιβάλλον (localhost).

Το project αυτό στοχεύει στην **απλοποίηση της διαδικασίας υπογραφής**, διατηρώντας ταυτόχρονα τη συμβατότητα και τη λειτουργικότητα του επίσημου signotec WebSocket Pad Server.

---

## ✍️ Τι προσφέρει

- 📌 Λήψη ηλεκτρονικής υπογραφής από συσκευές **signotec** (π.χ. Sigma)
- 🌐 Χρήση μέσω web browser (Chrome, Edge, Firefox κ.ά.)
- 🖊️ Ζωντανή απεικόνιση υπογραφής σε πραγματικό χρόνο
- 🖼️ Εξαγωγή εικόνας υπογραφής
- 🔐 Παρουσίαση δεδομένων υπογραφής (Base64, RSA)
- 🇬🇷 **Πλήρης μετάφραση περιβάλλοντος στα ελληνικά**
- 🧩 Διατήρηση της αρχικής δομής και λειτουργικότητας του signotec API

---

## 🧠 Φιλοσοφία του project

Το project **δεν επανεφευρίσκει** τον μηχανισμό της signotec.  
Αντίθετα:

- ✅ κρατά **απαράλλακτη τη βάση του επίσημου API**
- ✅ δεν πειράζει JavaScript functions, IDs ή WebSocket λογική
- ✅ επικεντρώνεται σε:
  - μετάφραση
  - καθαρότερο UI
  - καλύτερη αναγνωσιμότητα
  - προσαρμογή σε πραγματικά σενάρια χρήσης

Ιδανικό για:
- τοπικές επιχειρήσεις
- intranet λύσεις
- εκπαιδευτικούς σκοπούς

## ⚙️ Απαιτήσεις

- Windows ή Linux
- Εγκατεστημένο **signotec WebSocket Pad Server**
- Συνδεδεμένη συσκευή signotec (USB ή Serial)
- Σύγχρονος web browser

> ⚠️ Η εφαρμογή προορίζεται για **τοπική χρήση (localhost)**.  
> Δεν προτείνεται έκθεση σε δημόσιο δίκτυο χωρίς επιπλέον ασφάλεια.

---

## 🚀 Εκκίνηση

1. Εκκίνησε τον **signotec WebSocket Pad Server**
2. Άνοιξε το αρχείο HTML μέσω browser (ή localhost server)
3. Πάτησε **«Υπογραφή»**
4. Υπόγραψε στη συσκευή signotec
5. Η υπογραφή εμφανίζεται άμεσα στην οθόνη

---
